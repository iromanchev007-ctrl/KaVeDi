# KaVeDi — сайт студии тонировки и бронепленки

Современный адаптивный одностраничный сайт студии **KaVeDi** (Саратов)
с онлайн-записью. Заявки с формы приходят менеджеру прямо в Telegram.

## Состав проекта

```
kavedi-site/
├── index.html       — разметка сайта
├── styles.css       — стили (тёмная минималистичная тема)
├── script.js        — логика: форма, отправка в Telegram, анимации
├── api/send.js      — (опц.) серверная функция для безопасной отправки
├── vercel.json      — конфиг для Vercel
├── netlify.toml     — конфиг для Netlify
├── assets/          — изображения (см. ниже)
└── README.md
```

## ⚙️ Что нужно вставить самому

### 1. Telegram-бот (обязательно)

Откройте файл **`script.js`** — в самом верху есть блок настроек:

```js
const TELEGRAM_BOT_TOKEN = "ВСТАВЬТЕ_ТОКЕН_БОТА";
const TELEGRAM_CHAT_ID   = "ВСТАВЬТЕ_CHAT_ID";
```

- **TELEGRAM_BOT_TOKEN** — создайте бота у [@BotFather](https://t.me/BotFather)
  командой `/newbot` и скопируйте токен.
- **TELEGRAM_CHAT_ID** — узнайте свой ID у бота
  [@userinfobot](https://t.me/userinfobot). Для группы добавьте туда вашего
  бота и используйте ID группы (начинается с `-100…`).

После этого форма сразу заработает.

### 2. Email (по желанию)

Замените `XXX@mail.ru` на реальный адрес в `index.html`
(блоки «Связь» и «Подвал»).

### 3. Картинки

Положите свои фото в папку `assets/` со следующими именами:

| Файл                     | Где используется            |
|--------------------------|-----------------------------|
| `assets/hero.jpg`        | фон главного экрана (авто)   |
| `assets/tonirovka.jpg`   | карточка «Тонировка авто»    |
| `assets/syomnaya.jpg`    | карточка «Съёмная тонировка» |
| `assets/bronirovanie.jpg`| карточка «Бронирование кузова»|
| `assets/optika.jpg`      | карточка «Бронирование оптики»|

Если картинок нет — сайт всё равно работает: вместо фото
показываются аккуратные тёмные заливки.

## 🔐 Безопасный вариант (токен скрыт)

Прямая отправка из браузера показывает токен бота в коде страницы.
Чтобы спрятать его на сервере:

1. В `script.js` поставьте `const USE_SERVER_ENDPOINT = true;`
2. На хостинге добавьте переменные окружения:
   `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`.
3. Функция `api/send.js` сделает всё остальное.

## 🚀 Публикация

### Vercel
1. Загрузите проект на GitHub.
2. На [vercel.com](https://vercel.com) → **Add New → Project** → выберите репозиторий.
3. Framework: **Other**, остальное по умолчанию → **Deploy**.

### Netlify
1. Загрузите проект на GitHub.
2. На [netlify.com](https://netlify.com) → **Add new site → Import an existing project**.
3. Publish directory: `.` → **Deploy**.

Готово — сайт онлайн.

## Особенности

- Полностью адаптивный (ПК, планшет, смартфон).
- Плавная анимация появления блоков при прокрутке.
- Hover-эффекты для кнопок и карточек.
- SEO-разметка (meta, Open Graph, Schema.org).
- Выпадающие списки марок и моделей авто с возможностью ввода вручную.
- Встроенная Яндекс.Карта с меткой на адресе.
